레시피 추천

현재 냉장고에 있는 재료들을 토대로 레시피를 추천합니다.

상세 레시피

해당 레시피의 상세 정보를 확인할 수 있습니다.

일기 기능

음식을 직접 만든 사진과 함께 작게 일기를 쓸 수 있습니다.

적용한 기술


메인 화면 - TableView 안에 CollectionView를 넣어 조건에 맞는 리스트의 정보를 filter를 이용해 거른 후, dequeueReusableCell 메소드를 사용하여 화면에 반영함

재료등록 - TableView의 설정 중, accessoryType의 checkmark를 사용하여 선택된 재료를 체크표시로 볼 수 있게 만듬

레시피 - filter를 이용하여 각 조건에 맞는 그룹을 생성한 후, prepare function을 사용해 각각의 레시피에 맞는 상세 레시피를 segue로 이어줌

레시피 검색 - UISearchBar를 사용하여 검색창의 텍스트를 입력 받아 filter로 걸러 리스트를 제공할 수 있게 만듬.

다이어리 - CollectionView를 사용해 메인 화면을 구성하고 글쓰기 화면은 TableView에 Textfield를 넣어 제목과 내용을 입력받아 CollectionViewCell에 받아오게 만듬
